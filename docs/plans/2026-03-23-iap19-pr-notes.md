## Escopo

### Implementado
- Rota `/` convertida para login real.
- Rota `/superadmin` protegida por sessao/role.
- Rota `/mock/legacy-chat` criada com a tela mock antiga (mock legado removido da home).
- Rota placeholder `/app` para CTA do bloqueio `tenant_user`.
- Integracao web/mobile feita via `@repo/convex-backend` + `convex/react`.
- Painel superadmin funcional (Tenants, Usuarios, WABA, IA).
- Logout funcional com limpeza de sessao.
- Nenhuma extensao no `packages/backend`; pacote legado permanece fora do runtime oficial.

### Fora de Escopo
- Persistencia em banco externo (store atual continua in-memory).
- Fluxo operacional tenant_user em `/app` (apenas placeholder).
- Observabilidade aprofundada (somente base minima da fase atual).

## Riscos

1. Estado in-memory nao sobrevive a restart do processo (esperado no estagio atual).
2. Se houver futura exposicao HTTP dedicada, contratos devem continuar desacoplados de `packages/backend`.
3. Sem e2e automatizado completo; cobertura principal esta em testes unitarios/integração de backend + fluxo auth web.
